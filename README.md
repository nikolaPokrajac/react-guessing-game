# React Guessing Game

Check out [Live Demo](https://react-guessing-game.netlify.app/)

### Getting Started

There are two methods for getting started with this repo.

#### Familiar with Git?

```
> git clone git@github.com:nikolaPokrajac/react-guessing-game.git
> cd react-guessing-game
> npm install
> npm run dev
```

#### Not Familiar with Git?

Click [here](https://github.com/nikolaPokrajac/react-guessing-game.git) then click on "Clone or download" and download the .zip file. Extract the contents of the zip file, then open your terminal, change to the project directory, and:

```
> npm install
> npm run dev
```
